## Getting Started

Install the module with: `npm install lsa-ingest`

```js
var lsa-ingest = require('lsa-ingest');
lsa-ingest.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g lsa-ingest
$ lsa-ingest --help
$ lsa-ingest --version
```
